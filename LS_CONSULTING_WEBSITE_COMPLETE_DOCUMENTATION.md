### DATA SCALE 섹션
**배경색**: Navy 50

**통계 4개**:
| 숫자 | 설명 |
|------|------|
| 150,000+ | 월평균 분석 데이터 포인트 |
| 20+ | 암종별 세분화 분석 |
| 주간 | 리서치 업데이트 주기 |
| 5년+ | 누적 데이터 기간 |

---

### 분석 영역 (4가지)

1. **암종별 검색 패턴**
   - 폐암, 유방암, 대장암 등 각 암 종류별 환자의 정보 탐색 행동 분석

2. **치료 단계별 관심사 변화**
   - 진단, 수술, 항암, 회복 단계별 환자 질문 및 우려사항 추적

3. **증상 및 부작용 패턴**
   - 환자들이 가장 자주 호소하는 증상과 부작용의 시간대별 분포

4. **병원 선택 의사결정 요인**
   - 환자가 병원을 선택할 때 중요하게 고려하는 요소의 우선순위

---

### 분석 방법론
**배경색**: Navy 900

**4단계 프로세스**:

1. **데이터 수집** (Database 아이콘)
   - 온라인 커뮤니티, 검색 데이터, 환자 포럼 등 다양한 채널에서 체계적 수집

2. **AI 자연어 분석** (Search 아이콘)
   - 자연어 처리 기술로 환자 질문과 감정, 의도를 자동 분류 및 분석

3. **트렌드 추적** (TrendingUp 아이콘)
   - 시간대별, 계절별 환자 니즈 변화 패턴을 모니터링하고 예측

4. **인사이트 도출** (BarChart3 아이콘)
   - 데이터를 전략적 실행 가능한 인사이트로 전환하여 제공

---

### 분석 사례 (3가지)

**사례 1: 항암 3회차 시점의 불안 급증**
```
항암치료 3회차 전후로 "계속해야 하나", "효과가 있는 건가" 등의 
질문이 284% 증가. 이 시점에 환자 안심 프로그램과 치료 효과 
모니터링 커뮤니케이션이 필요함을 발견.
```

**사례 2: 병원 선택 시 '재활 프로그램'의 중요도 상승**
```
2023년 대비 2026년 병원 선택 기준에서 '재활 및 회복 프로그램' 
언급이 156% 증가. 단순 치료를 넘어 전체 회복 여정을 지원하는 
병원에 대한 니즈 확인.
```

**사례 3: 보호자 대상 정보 탐색의 분리**
```
환자 본인과 보호자가 찾는 정보의 내용과 타이밍이 다름을 발견. 
보호자는 경제적 부담과 장기 케어 방법에 더 집중. 
타겟별 맞춤 콘텐츠 전략 필요.
```

### 7-2. 병원 이탈 전략 리서치 (/research/pvm-hospital-switch)

**태그**: Healthcare Strategy Research

**제목**: 암 환자는 언제 병원을 바꾸는가
**부제**: 치료가 아니라 "관리 경험"에서 결정이 갈린다

**서브라인**: 2026년 2월 PVM 데이터 531건 기반 병원 이탈 요인 분석

**메타 정보**:
- 2026년 2월 분석
- 531건 데이터
- 12 min read

---

**핵심 발견 3가지**:

1. **치료 기술보다 "관리 경험"**
   ```
   환자는 치료 기술을 직접 평가하기 어렵지만,
   부작용 대응, 상담 품질, 정보 제공은 즉시 체감.
   관리 경험이 병원 선택의 핵심 기준.
   ```

2. **부작용 방치 = 신뢰 붕괴**
   ```
   "부작용을 말했는데 대수롭지 않다고 했어요"가 반복 패턴.
   부작용 호소에 대한 즉각적 대응 부재가 이탈 요인 1위.
   ```

3. **상담 로드맵의 부재**
   ```
   "다음에 뭘 해야 하나요?"에 대한 명확한 답이 없을 때
   환자는 다른 병원에서 "계획"을 찾으려 함.
   ```

---

**병원 이탈 요인 순위** (막대그래프):
1. 부작용 관리 불만 (38%)
2. 상담 로드맵 부재 (27%)
3. 정보 부족 (18%)
4. 대기 시간 (10%)
5. 거리/접근성 (7%)

---

**이탈 시점 분석** (라인그래프):
- 항암 1-2회차: 낮음
- 항암 3-4회차: 급증 ⬆️
- 항암 5회차 이후: 완만한 증가
- 치료 종료 후: 재상승 ⬆️

**인사이트**:
```
항암 3-4회차 시점에 부작용 누적과 불안이 정점.
이 시기 집중 상담이 이탈 방지의 핵심.
```

---

**요양·재활 병원의 대응 전략** (딥블루 박스):

1. **부작용 전담 상담 시스템**
   - 환자별 부작용 모니터링 프로토콜
   - 주간 체크인 시스템
   - 증상별 대응 가이드 제공

2. **치료 로드맵 시각화**
   - 단계별 예상 일정표
   - 각 단계별 목표와 체크포인트
   - 다음 단계 사전 안내

3. **정보 제공 체계화**
   - 자주 묻는 질문 사전 제공
   - 단계별 교육 자료

**설정**:
```javascript
<BarChart data={data}>
  <CartesianGrid strokeDasharray="3 3" stroke="#E5E7EB" />
  <XAxis dataKey="name" stroke="#666666" />
  <YAxis stroke="#666666" />
  <Tooltip />
  <Bar dataKey="value" fill="#0F2B46" radius={[8, 8, 0, 0]} />
</BarChart>
```

**2. 파이차트 (PieChart)**
- 암종별 분포
- 감정 패턴 분석

**설정**:
```javascript
<PieChart>
  <Pie
    data={data}
    cx="50%"
    cy="50%"
    labelLine={false}
    label={({ name, value }) => `${name} ${value}%`}
    outerRadius={100}
    dataKey="value"
  >
    {data.map((entry, index) => (
      <Cell key={`cell-${index}`} fill={entry.color} />
    ))}
  </Pie>
  <Tooltip />
</PieChart>
```
